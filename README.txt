AMELIORATEAI WEBSITE FILES
==========================

Files included:
1. index.html      - Home page
2. services.html   - Services page
3. contact.html    - Contact page
4. styles.css      - Website styling
5. script.js       - Mobile menu script

How to preview:
- Open index.html in your browser.
- Keep all files in the same folder.

How to customize:
- Replace contact@ameliorateai.com with your final Zoho email if different.
- Edit the service text in services.html.
- Edit color values in styles.css. Current design uses black background and blue accent fonts/buttons.

Contact form note:
The current contact form uses mailto, which opens the visitor's email app.
For a professional contact form, create a Zoho Form and embed it into contact.html.
